# CentralizedNDN

This system have 2 Phases :
1. Phase Bootstrap
   This Phase have 2 steps, there are:
   1. Hello Procedure
   2. Router Update
2. Phase Routing
   This Phase have 3 steps, there are:
   1. Prefix Registration
   2. Route calculation
   3. Route Installation
